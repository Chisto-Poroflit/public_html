Описание

Необходимо создать WEB приложение, позволяющее автоматизировать процесс создания расписания учебных занятий университета. Приложение должно учитывать:

  • личную загрузку преподавателей
  • числитель/знаменатель
  • перенос занятий после создания расписания
  • разграничение прав доступа пользователей по ролям и принадлежности к учебной группе

Приложение должно позволять администраторам создавать расписания занятий на основе учебных планов, количества и типа аудиторий (лекционная/практическая). Обычным пользователям - просматривать индивидуальное расписание. Предполагается выбрать локальное хранилище учетных записей пользователей. Как альтернатива - проверка подлинности Windows

Приложение должно реализовывать следующий функционал:

  • Загрузка информации о учебных планах (json)
  • Загрузка информации о аудиториях (json)
  • Загрузка информации о учебных группах (json)
  • Регистрация пользователей (в случае локального хранилища учетных записей)
  • Аутентификация пользователей
  • Авторизация пользователей
  • Составление расписания
  • Отправка расписания на печать
  • Сохранение расписания (json)
  • Загрзка старого расписания
  • Редактирование существующего расписания
  • Отображение индивидуального расписания группы
